iOS 26 Progress Bar

Современный прогресс-бар с эффектом liquid glass и анимациями в стиле iOS 26

Особенности дизайна

  • Liquid Glass Effect - Эффект стекла с размытием и насыщением цвета
  • Gradient Fill - Градиентная заливка с бликовыми эффектами
  • Dynamic Shine - Анимация бликов, движущихся по активным сегментам
  • Pulse Animation - Пульсация активных сегментов для визуальной обратной связи
  • Adaptive Design - Адаптивный дизайн для разных размеров экранов
  • Smooth Transitions - Плавные переходы между состояниями

Техническая реализация

Прогресс-бар использует современные CSS технологии:

/* Основные технологии */ - backdrop-filter: blur() saturate() - CSS градиенты и тени - CSS анимации и переходы - Flexbox для адаптивной верстки - CSS переменные для темизации

HTML структура

<div class="progress-container-ios26"> <div class="progress-segment-ios26"></div> <div class="progress-segment-ios26"></div> <div class="progress-segment-ios26"></div> </div>

Активация сегментов

// JavaScript для активации сегментов const segment = document.querySelector('.progress-segment-ios26'); segment.classList.add('active', 'pulse');

Инструкция по использованию

  1. Подключите CSS файлы в ваш HTML:
    <link rel="stylesheet" href="assets/styles/liquid-glass.css"> <link rel="stylesheet" href="assets/styles/ios26-progress.css">
  2. Добавьте HTML структуру прогресс-бара
  3. Используйте JavaScript для активации сегментов
  4. Настройте цвета через CSS переменные при необходимости
